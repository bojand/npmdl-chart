Or a specific range like /async?range=2018-01-01:2019-02-01

Chart API

/chart/{name}[.svg|.png]

The chart API can be used to directly generate the download counts chart image and embed it.
The image extension is optional and can be used to control the image format.
By default if omitted a SVG chart image is rendered.

Query parameters

  • range - the range parameter to query. See NPM download counts API documentation.
  • w - the width of the image.
  • h - the height of the image. If none provided the default aspect ratio is kept.

Examples

  • /chart/express - Default SVG image
  • /chart/express.svg - Optionally include image format extension explicetly
  • /chart/express.png - PNG render with all other settings being default
  • /chart/express?range=last-month - Specific range
  • /chart/express?range=last-month&w=600 - Specific range and width
  • /chart/express?range=2017-01-01:2017-11-01&w=600&h=300 - Specific range and dimensions
  • /chart/express.png?range=2017-01-01:2017-11-01&w=400&h=200 - Render PNG and explicitly set all options
